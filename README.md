# Media-de-idades
Atividade proposta no curso de Analise e Desenvolvimento de Sistemas do Instituto Federal de São Paulo.

Nessa atividade utilizamos estruturas de decisão e de repetição para implementar um programa que recebe uma quantidade n de idades informadas pelo usuario e retorna no final quantas idades foram inseridas, qual a maior e a menor idade inserida e qual a media de todas as idades que foram informadas.

# Estruturas utilizadas

- Do-while
- If-Else
